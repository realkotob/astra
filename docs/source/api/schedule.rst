Overview
--------

The ``schedule`` module provides functionality for reading and processing observation schedules for the Astra observatory automation system.

Key Functions
------------

- ``update_times``: Update schedule times based on a specified time factor
- ``process_schedule``: Process a schedule file into a usable DataFrame format

The schedule system allows for defining a sequence of observatory actions that are executed according to a timeline.
