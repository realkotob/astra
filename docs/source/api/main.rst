Overview
--------

The ``main`` module contains the entry point for the Astra application. It provides command-line interface functionality and handles starting the observatory automation system.

Key Components
-------------

- ``main``: Main entry point function that initializes and starts Astra
- Command-line argument parsing
- Observatory initialization and management
- Signal handling for graceful shutdown
