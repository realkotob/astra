Overview
--------

The ``config`` module provides configuration management for the Astra system. It defines paths and settings used throughout the application.

Key Components
-------------

- ``Config``: Main configuration class that manages paths and settings
- ``paths``: Property that provides access to standard file paths
- Configuration is loaded from standard locations and can be customized
