Overview
--------

The ``image_handler`` module provides functionality for handling astronomical images, including saving, processing, and organizing image data.

Key Functions
------------

- ``create_image_dir``: Create directory structure for storing images
- ``save_image``: Save camera image data to FITS files
- ``add_wcs_to_header``: Add World Coordinate System information to FITS headers
