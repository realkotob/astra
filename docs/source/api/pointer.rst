Overview
--------

The ``pointer`` module provides functionality for telescope pointing corrections based on astrometry. It helps improve the accuracy of telescope pointing by analyzing images and adjusting telescope coordinates.

Key Components
-------------

- ``PointingCorrectionHandler``: Class that handles pointing corrections
- ``PointingCorrection``: Class that represents a pointing correction calculation
- ``ImageStarMapping``: Class for mapping stars in images and performing astrometric calculations
