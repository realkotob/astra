Version 0.3.0 (2025-04-14)
------------------------

* Added autofocus functionality using the astrafocus library
* Improved pointing correction with enhanced star detection
* Added support for multiple camera binning modes
* Implemented automatic flat field sequence at optimal sky brightness
* Added pointing model building capability
* Enhanced autoguiding calibration
* Improved error handling and recovery
* Added detailed logging to database
* Various bug fixes and performance improvements

Version 0.2.0 (2024-11-15)
------------------------

* Added autoguiding capability using the donuts library
* Implemented observatory safety monitoring
* Added FITS header management and completion
* Improved schedule handling with more action types
* Enhanced dome control integration
* Added support for multiple observatory configurations
* Performance optimizations for image acquisition

Version 0.1.0 (2024-06-03)
------------------------

* Initial release
* Basic observatory control with ASCOM Alpaca
* Simple scheduling system
* Support for camera, telescope, and filter wheel control
* Basic image acquisition and storage
* Manual focusing support
